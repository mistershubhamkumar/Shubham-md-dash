## 🎯 Features

- 🎨 **Modern UI** with hover animations
- ⚙️ **Panel Code** popup with copy button
- 🔑 **Session ID** link (open in new tab)
- 🔌 **Dynamic Plugins** from `plugins.txt`
- 🔐 **Add Plugin Option** (password protected)
- 🧪 **ENV Variables** with copy examples
- 💬 **FAQ Accordion** in HTML
- 📱 Fully responsive (mobile/tablet/desktop)
- 🔗 **Direct WhatsApp Idea Request**

---

## 🔘 Panel Buttons Preview

| Button        | Action                                        |
|---------------|-----------------------------------------------|
| 🧩 Panel Code | Opens a tab with bot code to copy             |
| 🧿 Session ID | Redirects to auth/session manager             |
| 🔌 Plugin     | Lists all plugins with description + copy     |
| 🔐 ENV        | Shows env variables with sample values        |
| 💬 Request    | Opens WhatsApp chat                           |
| ❓ FAQ         | Expand/collapse question/answer accordion     |
